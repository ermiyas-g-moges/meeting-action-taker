# Meeting Action Taker

A Tauri desktop app that records meeting audio, extracts action items, and creates tasks in Notion or Trello.

## What is included

- React + Vite frontend
- Tauri Rust backend

## Setup

### 1. Install prerequisites

- Node.js
- Rust toolchain
- Tauri prerequisites for Windows: Visual Studio Build Tools with "Desktop development with C++"
  
### 2. Install dependencies

Open terminal in the project folder and run:

```terminal
npm install
```

### 3. Run the app

```powershell
npm run tauri:dev
```
